## 🚀 Quick Start

### Prerequisites
- **Node.js** 18.17.0 or higher
- **npm** 9.0.0 or higher

### Installation & Setup

\`\`\`bash
# Clone the legendary portal
git clone https://github.com/welshDog/tHe-HYPER-dOoK-STorY.git
cd tHe-HYPER-dOoK-STorY

# Install dependencies with turbo power
npm install

# Launch your portal in development mode
npm run dev

# Open your browser to http://localhost:3000
# 🎊 Welcome to your HYPER dOoK Portal!
\`\`\`

### Production Deployment

\`\`\`bash
# Build for production
npm run build

# Deploy to GitHub Pages
npm run deploy

# Or export static files
npm run export
\`\`\`

## 🎨 **ADHD-Optimized Design System**

### Color Psychology
- **Focus Blue** (`#00D4FF`): Attention and concentration
- **Success Green** (`#00FF88`): Achievement and dopamine

- **Themes**: Modify `tailwind.config.js` for custom color schemes
- **Fonts**: Add new dyslexia-friendly fonts in `globals.css`
- **Sounds**: Replace celebration audio files in `/public/sounds/`
- **3D Models**: Customize crystal shapes in `Crystal3DView.tsx`
